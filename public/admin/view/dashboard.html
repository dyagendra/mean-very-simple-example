Create main menu & sub menu & sub menu under 'sub menu'

Go to the ADD MENU option from left below, then put the main menu name under the top ‘Create New Main Menu As Per New Page’ input box, it will visible in your web site as main menu.


Every main menu Carrey a + option to create under itself a sub menu. When you create a sub menu the main menu link will be null.Every sub menu Carrey a + option to create under itself a sub menu. When you create a sub menu the sub menu under main menu, link will be null. Every menus have activate and deactivate option to visible or hide in web site, but it will be deleted,
